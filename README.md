# 王楚钦赛后微博舆情分析项目

本项目是“社会计算与生成决策智能”课程实践的 Lab 1 成果，围绕乒乓球运动员王楚钦在重要比赛
赢球或输球后的微博舆情开展小规模案例研究。

项目包含微博数据采集程序、8 场比赛的事件与关键词配置、原始采集记录、数据清洗脚本、质量报告，
以及可供后续情感与观点分析使用的帖子和评论数据。

## 目录

- `MediaCrawler`：微博采集程序、Lab 1 配置、原始数据和处理代码；
- `王楚钦舆情分析_Lab1`：整理后的课程交付结果。

推荐从 [`王楚钦舆情分析_Lab1/README.md`](王楚钦舆情分析_Lab1/README.md) 开始阅读。

## 当前数据

- 清洗后完整微博正文：74 条；
- 均衡微博正文：45 条（赢球 23、输球 22）；
- 赛后 24 小时内唯一一级评论：130 条；
- 噪声过滤后评论：118 条；
- 均衡评论比较集：61 条（赢球 30、输球 31）。

## 隐私与使用范围

数据中的用户标识已哈希，昵称已脱敏，不保存头像、主页、性别、IP 归属地等可定位个人的信息。
微博历史搜索和热门评论接口不是完整档案，因此结果只适用于课程中的小规模案例比较，不能代表微博总体舆情。

采集程序仅限学习和研究使用。使用前请阅读 [`MediaCrawler/LICENSE`](MediaCrawler/LICENSE)，遵守平台条款、
robots.txt、合理请求频率和相关法律法规。
