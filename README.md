# 王楚钦赛后微博舆情分析项目

本项目是“社会计算与生成决策智能”课程实践，围绕王楚钦在重要比赛赢球或输球后 24 小时内的微博舆情开展小规模案例研究。完整流水线依次为：

1. **Lab 1 数据构建**：配置 8 场比赛，采集、清洗、脱敏并形成均衡正文与一级评论样本；
2. **Lab 2 情感与观点分析**：按正文／评论、单场与胜负组生成领域情绪、议题、代表观点和质量报告；
3. **Lab 3 三合一决策助手**：基于可追溯证据包提供自动简报、证据约束问答、三方案建议及人工确认后的 Markdown 导出。

## 目录与阅读入口

- `MediaCrawler`：微博采集程序及其许可证；
- [`王楚钦舆情分析_Lab1/README.md`](王楚钦舆情分析_Lab1/README.md)：数据、研究范围与清洗质量；
- [`王楚钦舆情分析_Lab2/README.md`](王楚钦舆情分析_Lab2/README.md)：情感分析方法、结果与可复现流程；
- [`王楚钦舆情分析_Lab3/README.md`](王楚钦舆情分析_Lab3/README.md)：三合一决策台、离线模式、演示和测试。

## 当前数据

- 清洗后完整微博正文：74 条；
- 均衡微博正文：45 条（赢球 23、输球 22）；
- 赛后 24 小时内唯一一级评论：130 条；
- 噪声过滤后评论：118 条；
- 均衡评论比较集：61 条（赢球 30、输球 31）。

Lab 2 和 Lab 3 的核心分析输入即 45 条均衡正文与 61 条均衡评论；正文和评论按各自口径分开统计，不直接混合比例。

## 启动 Lab 3

先按 [Lab 3 README](王楚钦舆情分析_Lab3/README.md) 使用 Python 3.11 安装依赖，然后在仓库根目录运行：

```powershell
.\.venv\Scripts\python.exe -m streamlit run .\王楚钦舆情分析_Lab3\app.py
```

DeepSeek API key 可选；没有 key 或在线调用失败时，三项核心功能会自动使用确定性离线结果。所有方案仅供定性比较，不进行效果预测，也不会自动发布，最终决定由人工复核。

## 隐私与使用范围

数据中的用户标识已哈希，昵称已脱敏，不保存头像、主页、性别、IP 归属地等可定位个人的信息。
微博历史搜索和热门评论接口不是完整档案，因此结果只适用于课程中的小规模案例比较，不能代表微博总体舆情。新加坡输球事件只有 1 条可用评论，中国大满贯赢球事件没有可用评论，Lab 3 会持续显示这两项覆盖警告。
