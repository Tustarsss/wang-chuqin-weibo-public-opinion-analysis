# 王楚钦赢球／输球后微博舆情：Lab 1 交付结果

本目录是课程项目 Lab 1 的整理结果。研究对象为 8 场重要单打比赛，比较比赛结束后
24 小时内的微博正文与一级评论。数据适合课程中的小规模案例比较，不代表微博总体舆情。

## 推荐交给 Lab 2 的数据

- 帖子：`01_数据/wang_chuqin_weibo_balanced.csv`，45 条（赢球 23、输球 22）；
- 评论：`01_数据/wang_chuqin_weibo_comments_balanced.csv`，61 条（赢球 30、输球 31）。

帖子和评论应依据 `content_type` 分开统计情感分布，再比较两类来源的结果，不建议直接混合计算比例。

## 数据文件说明

- `wang_chuqin_weibo_clean.*`：清洗后的完整帖子，74 条；
- `wang_chuqin_weibo_balanced.*`：每场最多 6 条的帖子比较集，45 条；
- `wang_chuqin_weibo_comments_24h.*`：时间窗内全部唯一评论，130 条；
- `wang_chuqin_weibo_comments_analysis_ready.*`：排除明显泛互动噪声后的评论，118 条；
- `wang_chuqin_weibo_comments_balanced.*`：每场最多 10 条、跨锚点轮取的评论比较集，61 条。

CSV 便于 Excel 查看和课程分析，JSONL 便于后续 Python 流水线读取。

## 目录结构

- `01_数据`：帖子和评论数据；
- `02_质量报告`：采集、过滤、去重及分组数量；
- `03_说明与配置`：研究设计、事件时间窗和关键词；
- `04_复现脚本`：帖子及评论清洗程序。

## 主要局限

微博移动端历史搜索与热门评论接口不是完整档案。新加坡输球事件只有 1 条可用评论，
中国大满贯赢球事件没有通过噪声过滤的评论，因此不适合对每一场比赛单独下结论。
